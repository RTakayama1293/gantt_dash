# EEZO 2026 タスクダッシュボード

EEZO（北海道食材EC）の2026年度週次タスクを可視化するインタラクティブなガントチャートダッシュボードです。

## 概要

- **240タスク**の年間計画を視覚化
- **四半期（Q1〜Q4）**× **週（W1〜W12）**のマトリクス構造
- **2名の担当者**（松永・高山）× **5カテゴリ**のタスク管理

## 主な機能

### ガントチャート
- 📊 インタラクティブなタイムライン表示
- 🎨 カテゴリ/担当者別の色分け
- 🔍 ホバーでタスク詳細表示
- ⭐ マイルストーンの強調表示

### フィルタリング
- 四半期選択（Q1〜Q4）
- 担当者選択（松永/高山）
- カテゴリ選択（5種類）
- 日付範囲スライダー

### 表示設定
- 期間粒度切替（日/週/月）
- グループ化（担当者別/カテゴリ別）
- ソート順変更

### エクスポート
- 📁 Excel形式でダウンロード
- フィルター適用後のデータを出力

## クイックスタート

## データ形式

### CSV構造

| カラム | 説明 |
|--------|------|
| 四半期 | Q1〜Q4 |
| 週番号 | W1〜W12 |
| 開始日 | タスク開始日 |
| 終了日 | タスク終了日 |
| 担当者 | 松永 or 高山 |
| カテゴリ | 5種類のカテゴリ |
| タスク | タスク内容 |
| 成果物/マイルストーン | 期待される成果物 |

## 開発

### Claude Code on the Web での開発

1. このリポジトリをGitHubにpush
2. https://claude.ai/code にアクセス
3. リポジトリを選択してセッション開始
4. 「ガントチャートアプリを実装して」と指示

### ローカル開発

```bash
# 仮想環境作成
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 依存関係インストール
pip install -r requirements.txt

# 開発サーバー起動（ホットリロード有効）
python src/app.py
```
